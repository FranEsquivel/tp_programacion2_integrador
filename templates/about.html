Sobre el sistema

Sistema de Gestión de Estudiantes desarrollado con Flask y PostgreSQL.

Descripción del sistema

Aplicación web CRUD desarrollada para la gestión de estudiantes, permitiendo registrar, editar, eliminar y buscar información académica mediante una interfaz moderna y responsive.

El sistema fue diseñado con enfoque en experiencia de usuario, validaciones de datos y organización eficiente de la información.

Funciones implementadas

  • Registro de estudiantes
  • Edición y eliminación de datos
  • Búsqueda avanzada por múltiples campos
  • Ordenamiento dinámico de registros
  • Validaciones frontend y backend
  • Mensajes interactivos del sistema
  • Diseño responsive con estilo futurista

Tecnologías utilizadas

  • Python
  • Flask
  • PostgreSQL
  • HTML5
  • CSS3
  • Docker
  • pgAdmin
